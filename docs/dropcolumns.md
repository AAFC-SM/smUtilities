# dropcolumns Function to filter the columns in the dataframe

```r
dropcolumns(dfname, colslist)
```

## Arguments

- `dfname`: Dataframe
- `colslist`: List of column names to drop

## Returns

Dataframe without those columns

## Description

dropcolumns Function to filter the columns in the dataframe

## Examples

```r
temp1 <- dropcolumns(df, c("One", "Two", "Three"))
```
